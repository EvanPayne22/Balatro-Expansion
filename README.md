# Balatro-Expansion
### Includes new jokers that focus on balancing card modifciations and hand types.

## New Jokers
### Lucky Day
#### Played  <span style="color:#424e54">Clubs</span> each give <span style="color:#FE5F55">X2</span> Mult when scored

### The Wild West
#### Each played <span style="color:#8867a5">Wild Card</span> gives between <span style="color:#FE5F55">X1.0</span> and <span style="color:#FE5F55">X3.0</span> Mult when scored

### Domino Effect
#### Each played <span style="color:#8867a5">Stone Card</span> retriggers when scored

### Jackpot
#### <span style="color:#F009dff">+777</span> Chips if played hand contains at least three <span style="color:#eac058">7s</span>
